# Ultra Short Summary

A project to rewrite the core Laboratory Information System (LIS) to improve performance, automate decisions, and ensure regulatory compliance.

**MVP Points**
- The system shall retain all existing core LIS functionalities.
- Admin users shall be able to create/add new users and associate them with roles.
- The system shall provide online help accessible from every screen.
- The system shall maintain existing HIPAA compliance capabilities.

**Key Constraints**
- Development must use the .NET 3.5 platform and SQL Server 2008 database.
- Production updates are restricted to scheduled Tuesday maintenance windows.
- All new code must follow defined coding standards and undergo code review.
