# Ultra Short Summary
A prototype smart home system for managing home environment, security, and appliances via web interface to support business decisions about future commercial development.

- Monitor and control temperature, humidity, security, and appliances through web interface
- Provide monthly planning and reporting capabilities for environmental parameters
- Support user authentication and data encryption for security
- Operate in simulated environment with all sensors and controllers simulated

- Must be completed within twelve months with five engineers
- Must use widely available hardware/software technology and standards
- Must minimize costs of all system elements
