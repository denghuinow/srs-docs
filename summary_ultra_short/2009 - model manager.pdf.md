# Ultra Short Summary
The Model Manager is a software tool for configuring, scheduling, running, monitoring, and stopping weather and climate model jobs across multiple clusters.

- Users can set up and submit new model or post-processing jobs through a Job-Setup module.
- The system supports submitting custom "by-hand" jobs and loading job configurations from files.
- Users can retrieve, modify, and re-run previously saved job configurations.
- The system provides monitoring of scheduled, running, and past jobs with options to stop, restart, or resume.

- The tool must manage jobs across one or more clusters with centralized node allocation.
- It must support both standard and custom input data sources and processing scripts.
- The system must be accessible via a web-based GUI and a command-line tool.

- The level of customization for post-processing and specific model options remains undecided.
