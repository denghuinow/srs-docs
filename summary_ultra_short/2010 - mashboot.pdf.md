# Ultra Short Summary
Mashbot is a web service for managing a company's social media presence by unifying interfaces and enabling scheduled marketing campaigns.

**MVP points**
- Schedule content for concurrent publishing across multiple social networks.
- View and compare historical campaign performance metrics.
- Create and manage user accounts with role-based permissions.
- Associate Mashbot accounts with external service accounts for authentication.

**Key constraints**
- The server requires no more than 1GB of RAM.
- Communication between client and server must use TLS encryption.
- The system must support integration with existing database software.
