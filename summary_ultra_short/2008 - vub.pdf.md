# Ultra Short Summary
A Publications Management System (PMS) for managing bibliographical information and full-text publications with web and command-line access.

- **MVP points**
  - Users can search, view, and download publications with role-based access control.
  - Members can register, log in, and manage personal account details.
  - Publishers can upload, edit, and bulk upload publications in supported formats.
  - Administrators can manage users, groups, and system configuration settings.

- **Key constraints**
  - The system must run on Linux, specifically the Wilma web server.
  - The web interface must comply with W3C XHTML 1.0 Transitional standards.
  - Only free software and Ruby programming language can be used.

- **Major risks/undecided issues**
  - Maximum file size for publications remains to be determined.
  - Performance requirements are not specified.
