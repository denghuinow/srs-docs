# Ultra Short Summary
A 3D visualization tool for biological data analysis to enable researchers to investigate and compare data attributes in a three-dimensional environment.

- MVP points
  - Plot data points in a scientifically correct manner with labeled axes and color coding by cluster.
  - Allow user interaction to rotate, zoom, and move the grid for orientation changes.
  - Display point names on single click and full details on double click.
  - Support up to 2000 data points from a formatted tab-delimited input file.

- Key constraints
  - Must run on both Windows and Linux operating systems.
  - Must be developed and ready for inclusion in a scientific paper by March 2005.
  - Execution must be kept on the server side.
