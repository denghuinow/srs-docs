# Ultra Short Summary
GParted is a graphical partition editor for creating, reorganizing, and deleting disk partitions, running as a LiveCD application that operates at system boot without installation.

- MVP points: Create, delete, resize, or move partitions while preserving data; format partitions to supported file systems; copy and paste partitions; check and repair file systems.
- Key constraints: Requires a working Linux environment or bootable media; does not support logical volume management (LVM2); depends on third-party libraries and file system tools.
- Major risks/undecided issues: Potential data loss from partition operations; Not mentioned.
