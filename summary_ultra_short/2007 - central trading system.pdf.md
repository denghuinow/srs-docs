# Ultra Short Summary
A subsystem of the Stock Trading System that completes stock trades by matching buy and sell instructions.

- **MVP points**
  - Process buy and sell instructions by matching them based on stock ID.
  - Save successful trade information to the Security Account Management subsystem.
  - Support querying trade information for statistical analysis and release.
  - Cancel trading instructions upon user request.

- **Key constraints**
  - Must handle high transaction frequency and system overhead.
  - Requires integration with multiple subsystems including Trade Client Serve and Trading Information Release.
  - Maintainers must be proficient in Java and socket programming.

- **Major risks/undecided issues**
  - Security issues are noted but not detailed.
