# Ultra Short Summary
A tool to assist analysts in goal and scenario mining, reconciliation, and management for privacy and security policy analysis.

- **MVP points**
  - Support four user access levels (administrator, project manager, analyst, guest) with distinct permissions.
  - Allow analysts to add, delete, and update goals and scenarios.
  - Enable project managers to manage privacy policies and assign analysts to projects.
  - Provide secure user authentication and password storage.

- **Key constraints**
  - The system must maintain traceability links between goals and source policies.
  - Access logs must be generated for all add, delete, and edit actions.
  - Security-related requirements are categorized as high priority.
