# Ultra Short Summary
Virtual-ED is a distance learning platform for NJIT that enables real-time communication, file management, and online testing.

**MVP Points**
- Provide instant messaging for real-time text communication between students and faculty.
- Support streaming audio and video for live lectures and conferences.
- Offer file hosting and management tools for assignments and collaboration.
- Include online testing capabilities with timed exams and file uploads.

**Key Constraints**
- The system must support only Internet Explorer, Firefox, and Safari browsers.
- The existing NJIT user database may only support 250 concurrent users.
- System rollouts and validations must be scheduled during low-usage periods.
