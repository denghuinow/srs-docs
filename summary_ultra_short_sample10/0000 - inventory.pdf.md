# Ultra Short Summary
A web-based Unified University Inventory System (UUIS) to integrate and manage inventory assets across three university faculties with secure access.

- **MVP points**
  - Users can create requests to borrow assets or reserve spaces.
  - Administrators can approve, reject, or delegate inventory requests based on hierarchical permissions.
  - Inventory administrators can add, edit, transfer, and return assets within authorized scope.
  - The system generates reports on assets, requests, and user permissions.
