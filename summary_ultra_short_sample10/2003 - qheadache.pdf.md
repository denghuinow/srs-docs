# Ultra Short Summary
A standalone puzzle game where players move blocks on a board to solve a specific headache.

- MVP points
  - Players can select, move, and arrange blocks on a 4x5 unit board.
  - The game supports undo and redo of the last thousand actions.
  - It tracks and displays player statistics including move count and completion time.
  - Players can save/load games and view top scores in a statistics window.

- Key constraints
  - The game requires a minimum 800x600 screen resolution and uses the Qt graphical library.
  - It must run on all operating systems supported by Qt, with portability to Windows OS.
  - Input is handled via mouse and keyboard, with sound being optional.
