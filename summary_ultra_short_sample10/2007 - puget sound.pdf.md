# Ultra Short Summary
A courseware system enhancement for University of Puget Sound to replace Blackboard with Moodle-based functionality.

**MVP Points**
- The system must allow for multiple file upload to be configured on a per-page basis.
- The system must allow actors to organize their voice clips into a portfolio for language courses.
- The system must allow web feeds to be turned on or off on a per-page basis.
- A course administrator must be able to grade an assignment with point ratios and feedback.

**Key Constraints**
- Enhancements must utilize existing Moodle APIs.
- The system must support at least 1000 concurrent users.
- The system should be available 24 hours a day, 7 days a week with 99% uptime.
