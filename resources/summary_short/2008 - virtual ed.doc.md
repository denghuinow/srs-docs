# Short Summary

- **Background and objectives**: Virtual-ED is a distance learning system designed for NJIT to improve communication between professors and students through a secure platform with features like messaging, audio/video tools, and online testing.
- **In scope**:
  - Instant messaging and group chat
  - Streaming audio and video (live and on-demand)
  - Customizable user profiles and interface
  - File hosting and management tools
  - Online testing with instant grading
- **Out of scope**:
  - Support for non-Microsoft, Apple, or Mozilla browsers
  - Languages other than English for system documentation
  - Integration with outdated or unsupported databases
  - Real-time collaboration in the first release of Virtual-Space
  - Not mentioned
- **Roles and core use cases**:
  - As a System Administrator, I want to manage the entire system and user enrollment so that the platform runs smoothly.
  - As an Administrative End User, I want to support my class and enrolled users so that learning activities are effective.
  - As a Limited End User, I want to customize my profile and access course materials so that I can participate in online learning.
- **Success metrics**:
  - Support for at least 250 concurrent users
  - 99% system uptime
  - User ability to access system on and off campus via broadband
- **Major constraints**:
  - Development must occur while university is in session with minimal downtime
  - Limited to Windows XP/Vista or Mac OS and specific browsers
  - Current database may only support 250 concurrent users
  - Users must have high-speed internet and recommended peripherals
  - System rollouts scheduled during low usage periods
