# Ultra Short Summary
- Triangulation Games is a Java-based application for playing and defining combinatorial triangulation games against AI or human players.

- MVP points
  - The system provides a graphical user interface supporting mouse and keyboard input.
  - Players can choose from multiple triangulation game types and opening positions.
  - Games support both human and AI players, including a default random AI.
  - Games are defined externally without modifying source code and end on predefined conditions.

- Key constraints
  - The software must run on Java 1.4 or later and be cross-platform compatible.
  - No platform-specific libraries or external databases are allowed.
  - The AI must calculate moves within 10 seconds.
