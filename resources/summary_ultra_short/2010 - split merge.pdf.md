# Ultra Short Summary
PDF Split and Merge is a free, open-source tool for manipulating PDF files through a graphical interface and command-line console.

- MVP points
  - Split PDF documents into chapters, single pages, or by size.
  - Merge multiple PDF documents or selected sections into one file.
  - Mix pages alternately from two PDF documents.
  - Rotate, reorder, or delete specific pages visually.

- Key constraints
  - Requires Java Runtime Environment version 1.6 or higher.
  - Licensed under GNU GPLv2.
  - Platform independent, runs on any system with a Java Virtual Machine.

- Major risks/undecided issues
  - Not mentioned.
