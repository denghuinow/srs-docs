# Ultra Short Summary
STEWARDS is a centralized data system providing one-point access to standardized watershed data from USDA-ARS research sites for conservation practice assessment.

**MVP Points**
- Store and manage diverse watershed data types including biophysical, management, and economic data.
- Provide web-based tools for browsing, querying, visualizing, and downloading data.
- Support metadata creation, search, and compliance with federal standards.
- Enable annual data uploads from watershed sites after local quality assurance.

**Key Constraints**
- Must use Microsoft SQL Server as the corporate standard database engine.
- Must comply with all USDA accessibility, web design, and security policies.
- Data will not be available to the public in real-time; updates occur annually.

**Major Risks/Undecided Issues**
- Funding dependency on NRCS beyond FY07 may impact timelines if reduced.
