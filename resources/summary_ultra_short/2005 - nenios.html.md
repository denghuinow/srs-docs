# Ultra Short Summary
A web-based child care management system to automate center operations and improve workflow efficiency.

**MVP Points**
- Role-based access for administrators, teachers, and assistants with different permissions.
- Track child attendance, immunizations, and teacher comments in a centralized database.
- Generate monthly billing with late-pickup fees and multi-child discounts.
- Provide printable reports including customer directory, enrollment status, and immunization history.

**Key Constraints**
- Must use Microsoft ASP.NET technology stack.
- Must support Internet Explorer and Netscape Navigator browsers.
- System must respond to user requests within 20 seconds.
