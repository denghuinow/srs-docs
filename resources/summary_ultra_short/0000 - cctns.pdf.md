# Ultra Short Summary
A centralized e-governance system to improve crime investigation and criminal tracking for Indian police and citizens.

- **MVP points**
  - Enable citizens to register complaints and interact with police.
  - Support investigation workflows and automate operational tasks.
  - Provide search functionality for cases, persons, and crime types.
  - Offer role-based navigation and task management for police personnel.

- **Key constraints**
  - The system must function in offline mode and low-bandwidth environments.
  - It must be developed using open standards and a service-oriented architecture.
  - Security measures must include SSL, VPN, and protection against SQL injection.
