# Ultra Short Summary
Project ZNIX is an object-based, application-independent information management framework designed to refine traditional systems by emphasizing information semantics and user-defined logical access.

**MVP points**
- Provide a centralized archive for user information from specified third-party applications.
- Enable hierarchical viewing and management of information through a browser GUI.
- Allow users to add, remove, modify, and archive information without disrupting the hierarchy.
- Expose functionality through a public API for creating, modifying, and querying Entities and relationships.

**Key constraints**
- Must run on Intel Pentium III or compatible 1-GHz PC with 256 MB RAM.
- Framework Shell must not utilize more than 40 MB of RAM.
- Must conform to Windows 2000 Logo Requirements and Unicode standards.
