# Ultra Short Summary

A web-based Supply Chain Management system for Ejada company to manage customer requests, supplier interactions, and product delivery coordination.

- **MVP points**
  - Coordinators can manage customer requests, items, and supplier relationships.
  - Customers can submit and track their product/service requests.
  - Suppliers can view requests and provide feedback on fulfillment capability.
  - System handles user authentication with role-based access for coordinators, customers, and suppliers.

- **Key constraints**
  - Must be web-based using ASP.NET and C# with MS SQL database.
  - Must integrate with Ejada's existing .NET framework.
  - Must support Internet Explorer and Firefox browsers.
