# Ultra Short Summary
A grid-based software infrastructure for bio-geo-chemical modeling using Daymet and Biome-BGC models with a graphical user interface.

- MVP points
  - Users can manage input data, run simulations, and manage output data.
  - The system provides a web-based portal for all user interactions.
  - The system supports Daymet and BiomeBGC modeling runs.
  - Users can visualize and download output data.

- Key constraints
  - The system must use the Globus toolkit.
  - The system must comply with NCAR security policies.
  - The system must use the NCAR Mass Storage System for all file-based storage.
