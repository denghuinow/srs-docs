# Ultra Short Summary
A plug-and-play USB-based web store system for new online store owners to quickly set up and manage e-commerce operations.

- MVP points
  - Manage customer accounts including registration, login, and profile storage.
  - Manage online store inventory with product categories and details.
  - Handle shopping cart functionality for item selection and checkout.
  - Process order confirmation with email notifications to customers.

- Key constraints
  - Must use SQL-based database for data storage.
  - Only compatible with specified browsers: Internet Explorer 6/7 and Netscape Communicator 4/5.
  - System must operate on Intel-based hardware with Slackware Linux and Apache web server.

- Major risks/undecided issues
  - Smooth transition from existing telephonic orders to the new online system.
  - Not mentioned.
