# Ultra Short Summary
PeaZip is a cross-platform file and archive manager providing a graphical interface for open-source compression utilities.

- Create archives in multiple supported formats with optional encryption.
- Extract contents from archives in various supported formats.
- Manage files with tools like secure deletion, splitting, and checksum calculation.
- Customize application behavior and interface through a settings menu.

- Must be developed using Delphi/Kylix, Object Pascal, or Pascal.
- Requires an x86-compatible CPU due to performance-critical ASM code.
- Distributed under the LGPL license as open-source software.

- Drag-and-drop from application to system is only supported on Windows.
