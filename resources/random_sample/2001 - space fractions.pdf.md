# Ultra Short Summary
A web-based educational game to improve sixth-grade students' fraction-solving skills through an interactive storyline.

- MVP points
  - An introductory movie sets up the storyline and can be skipped.
  - A main menu provides game access, help, and team information.
  - A series of multiple-choice fraction questions forms an adaptive storyline with scoring.
  - An ending scene displays the user's score and offers replay or exit options.

- Key constraints
  - The product must run in a web browser with Macromedia Flash 5 and JavaScript support.
  - All user input is restricted to mouse clicks; no keyboard input is required.
  - The game must be accessible over the Internet without permanent files on the user's computer.
