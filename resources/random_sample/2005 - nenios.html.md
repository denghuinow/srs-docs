# Short Summary

- **Background and objectives**: The Neñios Child Care Management (NCCM) software aims to automate daily operations of a child care center, improving workflow efficiency and enabling staff to focus more on child care through a centralized, web-based system.

- **In scope**:
  - Role-based access control for administrators, teachers, and assistants.
  - Child and parent account management including immunizations, attendance, and billing.
  - Daily reminders, classroom capacity tracking, and waiting list management.
  - Reporting features for customer directories, enrollment, and immunization status.
  - Late pick-up billing and multi-child discount calculations.

- **Out of scope**:
  - Federal, state, or local regulatory compliance processes.
  - Parent or guardian direct system access.
  - Mobile application or offline functionality.
  - Integration with external accounting or payment systems.
  - Automated communication with parents (e.g., emails or SMS).

- **Roles and core use cases**:
  - As an administrator, I want to manage customer accounts and generate reports so that I can oversee center operations and billing.
  - As a teacher, I want to record child behavior comments so that I can track development and support parent-teacher conferences.
  - As an assistant, I want to log child arrival and departure times so that I can track attendance and late pick-ups.

- **Success metrics**:
  - System responds to user requests within 20 seconds.
  - Supports up to 20 children per classroom and 100 entries in the waiting list.
  - All high-priority functional requirements (e.g., billing, security, immunization tracking) are implemented.

- **Major constraints**:
  - Must use Microsoft ASP.NET and a compatible web server.
  - Must support Internet Explorer and Netscape Navigator browsers.
  - User interface must have a blue background color.
  - Passwords must be 6–8 alphanumeric characters.
  - User sessions expire after 10 minutes of inactivity.
