# Short Summary

- **Background and objectives**: The Get Real website aims to attract and encourage Oregon high school students to pursue computer science degrees at Oregon University System campuses by providing teen-friendly, engaging content.

- **In scope**:
  - Redesign existing pages for improved readability and visual appeal.
  - Add new sections like High School Courses, Search, and RSS feeds.
  - Include video profiles and FAQ features for student engagement.
  - Support internal and external search functionality.
  - Ensure compatibility with major web browsers.

- **Out of scope**:
  - User documentation.
  - Special hardware, software, or communication interfaces.
  - Development assumptions or dependencies.
  - Expanding community and private college listings.
  - Knowledge base features (planned for Version 3).

- **Roles and core use cases**:
  - As a high school student, I want to explore CS careers and college options so that I can make informed decisions about my future.
  - As a web designer, I want clear style conventions and content priorities so that I can implement an effective, teen-friendly site.
  - As an OUS administrator, I want to track site usage and engagement so that I can evaluate and improve content over time.

- **Success metrics**:
  - Increased number of Oregon students choosing CS majors.
  - Positive feedback on site navigation and visual design.
  - Sustained or growing site traffic and time spent on pages.
