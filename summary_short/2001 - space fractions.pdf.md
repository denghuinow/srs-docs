# Short Summary

- **Background and objectives**: Space Fractions is a web-based educational game designed to improve sixth-grade students' fraction-solving skills through an interactive storyline, with score feedback and an umbrella system linking related math games.

- **In scope**:
  - Introductory movie setting up the game storyline.
  - Main menu with help section and game start option.
  - Multiple-choice fraction questions forming an adaptive plot.
  - Ending scene displaying score and plot conclusion.
  - Web-based question updater for administrators.

- **Out of scope**:
  - Keyboard input (mouse-only interaction).
  - Permanent local file storage for users.
  - Hardware-specific dependencies.
  - Integration with other educational software.
  - Multi-user simultaneous gameplay.

- **Roles and core use cases**:
  - As a student, I want to answer fraction questions in a game format so that I learn interactively.
  - As a teacher, I want to update game questions via a web interface so that I can customize content.
  - As a competitive student, I want to see my score and ranking so that I can track my progress.

- **Success metrics**:
  - Game runs in a web browser with Flash 5 and JavaScript support.
  - Students complete fraction problems with adaptive feedback.
  - Teachers can successfully update questions via the admin tool.
