# 2008 - virtual.md

## System Overview
VIRTUAL-EDU is an online learning system designed for NJIT, providing features including email, group chat, bulletin board, audio/visual tools, file hosting and management, customizable user profiles, and online testing with instant grading.^[1]^

## Key Requirements

### Functional Requirements
- **Instant Messaging**: High priority feature enabling real-time interaction between students and professors.^[2]^
- **Streaming Audio and Video**: Medium priority feature for audio and video calls, providing a real classroom experience.^[3]^
- **Customizable User Profile**: Medium priority feature allowing users to personalize their profiles with background colors, fonts, and layout.^[4]^
- **Virtual-Space**: Medium priority feature for file hosting and management, supporting document sharing and assignment submission.^[5]^
- **Online Testing (Virtual-Exam)**: High priority feature for real-time exam completion, combining multiple-choice, true/false, short answer, and essay questions.^[6]^

### External Interface Requirements
- **User Interface**: Designed with emphasis on user experience, compartmentalized by competency, allowing interface customization.^[7]^
- **Hardware Interfaces**: High-resolution display, keyboard, mouse, sound card, and voice input device required.^[8]^
- **Software Interfaces**: Seamless integration with local operating/hardware system, requiring Excel and PDF for report viewing.^[9]^
- **Communications Interfaces**: IE, Firefox, or Safari browsers, with pop-up and JavaScript enabled.^[10]^

### Non-Functional Requirements
- **Performance**: Minimum 500 MHz CPU, 256 MB RAM, broadband internet access recommended.^[11]^
- **Safety**: Users responsible for data backup, prohibited from uploading discriminatory or derogatory content.^[12]^
- **Security**: Compliance with local, state, and federal laws, mandatory password changes every three months.^[13]^
- **Software Quality**: Estimated 99% uptime, periodic maintenance may cause unavailability.^[14]^
